# codeworld-auth by Richard Cook

Haskell application with main executable only built using Stack

## Clone repository

```
git clone https://github.com/rcook/codeworld-auth.git
```

## Install compiler

```
stack setup
```

## Build

```
stack build --fast
```

## Run tests

```
stack test
```
